# Joan Didion Style

## 한줄 요약
절제된 문장으로 불안과 균열을 포착하는 냉정한 관찰자. 감정을 직접 말하지 않고, 디테일의 나열로 느끼게 한다.

## 핵심 특징
- **건조한 정밀함**: 감정적 상황을 감정 없이 묘사한다. 사실과 디테일을 정확하게 배치하면 감정은 독자 안에서 스스로 생긴다. 핵심 패턴: 구체적인 장소, 날짜, 사물의 이름을 나열하고 — 해석은 하지 않는다. 그 공백이 긴장을 만든다.
- **개인에서 보편으로**: "나"의 경험에서 출발하되, 어느 순간 시대의 공기로 확장된다. 단 "우리 모두"라고 직접 말하지 않는다.
- **열린 결론**: 답을 주지 않는다. 관찰을 끝까지 밀어붙이고 거기서 멈춘다. 독자가 불편해도 상관없다.
- **리듬의 반복과 변주**: 같은 구문을 약간씩 바꿔 반복하며 집요한 리듬을 만든다. 주술 같은 효과.
- **자기 의심**: 자신의 기억, 판단, 서술 자체를 의심하는 문장이 섞인다. "내가 이것을 제대로 보고 있는지 모르겠다."

## 자주 쓰는 기법
- 구체적 디테일의 나열 후 해석 없이 단락 종료 (공백이 의미를 만든다)
- 짧은 문장과 긴 문장의 교대 — 긴 문장은 하나의 관찰을 집요하게 풀어놓는다
- "나는 ~을 기억한다" 구문의 반복 변주
- 시간과 장소를 정확하게 박아넣기 ("1968년 여름, 하이트 애시버리에서")
- 관찰 중간에 자기 서술의 신뢰성을 흔드는 삽입구

## 피해야 할 것
- 감정을 직접 명명하기 ("슬펐다", "분노했다")
- 교훈이나 처방을 제시하는 결론
- 과도한 유머 (디디온은 재치가 있지만 웃기려 하지 않는다)
- 추상적이고 모호한 서술 (디디온의 힘은 구체성에 있다)

## 예시 문장 (스타일 모방)
- "그날 오후 3시, 서울역 대합실에서 나는 에스컬레이터가 멈춰 있는 것을 봤다. 왜 멈춰 있는지는 아무도 설명하지 않았다. 사람들은 멈춘 에스컬레이터를 계단처럼 걸어 올라갔다. 나도 그렇게 했다."
- "나는 왜 그것을 기억하는지 모른다. 정확히는, 왜 다른 것은 잊으면서 그것만 기억하는지 모른다. 기억이란 그런 것이다 — 라고 말하면 설명이 된 것 같지만, 아무것도 설명되지 않는다."
- "커피가 식어가고 있었다. 나는 그것을 알면서도 마시지 않았다. 마시지 않는 것에 특별한 이유는 없었다."

## 잘 맞는 주제
개인 체험과 시대의 교차, 상실과 변화, 장소의 의미, 자기 서술의 한계, 일상 속 균열
