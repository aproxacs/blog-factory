---
name: 고유 문체 개발 방향
description: 글쓰기 스타일 선호와 고유 톤 방향. 가독성 최우선, 특정 작가 고정 선호 없음, 건조한 관찰 + 슬쩍 웃기는 전환이 고유 톤.
type: feedback
---

사용자는 특정 작가 스타일을 고정 선호하지 않는다. 가독성과 선형 서사를 문장의 밀도보다 우선시한다. 러셀의 매 문단마다 비트는 패턴은 "산만하게" 느껴짐.

기존 작가 스타일 모방에서 점진적으로 고유 문체로 이동 중.

**Why:** 스타일 모방은 "잘 흉내 낸 글"의 한계에 갇힌다. AI 냄새를 줄이는 핵심은 대체 불가능한 개인 목소리. 외부 피드백(2026-03-31)에서도 "개인적 경험의 밀도 부족" 지적이 유효했음.

**How to apply:**
- 고유 톤: **건조한 관찰 + 슬쩍 웃기는 전환** (전자레인지 해동 버튼, 1조 달러 회계 버전, 스폰서드 마무리 등)
- "관찰"만이 아니라 "체험"의 밀도를 높일 것
- 판단 유보도 하나의 관점 — "모르겠다"가 과감한 단언보다 정직
- 스킬에 반영됨: 작가는 스타일에 충실, 편집장이 고유 톤으로 통합 (역할 분담)
- 스타일 추천 시 특정 스타일을 사용자 선호로 가중하지 말 것
