name: 메모리는 프로젝트 안에 저장
description: 메모리 파일을 프로젝트 .claude/memory/에 저장하고 심링크로 연결. 맥북 의존성 제거 목적.
type: feedback
---

메모리 파일은 프로젝트 내부 `.claude/memory/`에 저장한다. `~/.claude/projects/...` 경로는 이 디렉토리를 가리키는 심링크이다.

**Why:** 사용자가 맥북 의존성을 없애고 싶어함 (2026-04-01). 메모리를 git으로 커밋·공유할 수 있어야 다른 머신에서도 동일하게 작동.

**How to apply:** 메모리를 읽고 쓸 때 심링크가 해결해주므로 기존 경로 그대로 사용하면 됨. 새로운 메모리 파일도 동일 경로에 저장하면 자동으로 프로젝트에 반영된다.
